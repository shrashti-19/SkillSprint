# SkillSprint Backend

This is the backend for the SkillSprint project — a 21-day public/private challenge tracker.

## 🚀 How to Run

1. Install dependencies:
   ```bash
   npm install
   ```


2. Start server in dev mode:
   ```bash
   npm run dev
   ```
